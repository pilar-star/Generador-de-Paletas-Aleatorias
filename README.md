# Generador de Paletas Aleatorias
Este proyecto es una página web con la finalidad de crear, según formato (HSL o HEX) y cantidad de colores (6, 8 o 9), una paleta de colores totalmente aleatoria.

## Modo de Uso ⚙️
1. Elegir la cantidad de colores deseada (6, 8 o 9 colores)
2. Elegir el formato en que aparecerá el nombre de los colores (HSL: Hue, Saturation y Lightness o HEX: hexadecimal)
3. Hacer click sobre el botón "Generar Paleta"

De esta manera aparecerá instantáneamente una paleta con el nombre del formato y la cantidad de colores elegidos de manera aleatoria. 
El formato puede ser cambiado indistintamente sin que cambien los colores ni tener que tocar el botón "Generar Paleta" nuevamente.
En caso de querer cambiar la cantidad de colores, si es necesario tocar el botón "Generar Paleta". 

### Herramientas Utilizadas 🛠️
Para este proyecto fue utilizado esclusivamente Visual Studio Code para escribir el código, la consola del navegador de google para reconocer ubicaciones de problemas y el navegador de google para abrir el proyecto y obtener una previsualización a medida que se realizaban actualizaciones en el código.
